Includes implementation of following algorithms from scratch:

* Linear Regression
* Logistic Regression
* Decision Trees
* Random Forest
* Naive Bayes
* K-Nearest Neighbours

The package also includes functions to compute model performance metrics:
* MSE
* RMSE
* AUC
* ROC
* Confusion Matrix
* Precision,
* Recall
* F1 score
* Accuracy
